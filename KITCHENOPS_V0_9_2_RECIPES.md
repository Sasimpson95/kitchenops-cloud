# KitchenOps v0.9.2 — Recipes

This release strengthens the recipe engine on top of v0.9.1.

## Included
- Recipe categories
- Search by recipe name, category and allergen
- Product ingredients with recipe units
- Preparation/component recipes
- Live total cost and cost per yield/portion
- Selling price, food-cost %, GP value and GP % for menu items
- Target GP and target selling price
- Method steps
- Allergens
- Active/archive handling
- Improved empty state
- Recipe costing settings and history included in the per-business browser workspace

## Notes
No Supabase migration is required for this release. Existing recipes without a category automatically display as Uncategorised.
