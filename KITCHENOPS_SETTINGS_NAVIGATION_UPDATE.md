# KitchenOps Settings Navigation Update

## What changed

The Operations sidebar now keeps only day-to-day operational modules plus one Settings link.

Moved out of the main sidebar:

- Reserva
- POS Integration
- Sites
- Users
- Audit Log

These are now reached through the Settings hub.

## New routes

- `/settings` — settings hub
- `/settings/business` — business details and stocktake schedule
- `/settings/sites` — site management
- `/settings/users` — staff management
- `/settings/integrations` — Reserva and POS
- `/settings/audit-log` — audit activity

The original routes remain available so existing links and bookmarks continue to work.

## No SQL required

This update changes navigation and page organisation only.
