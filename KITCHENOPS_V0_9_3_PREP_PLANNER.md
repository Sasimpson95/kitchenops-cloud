# KitchenOps v0.9.3 — Prep Planner

## Included
- Tomorrow-first manager planning workflow.
- Recipe Library search and recipe preview from the prep plan.
- Planned quantities can be edited or removed while still planned.
- Chef completion from the Home screen with actual produced quantity.
- Manager approval updates stock using the approved quantity.
- Remaining prep can be explicitly added to tomorrow.
- Tomorrow automatically becomes today on the next local calendar day.
- Previous day prep is automatically archived into Prep History.
- History records planned, produced, status, chef and approving manager.
- Zero-site businesses get a safe Create First Site empty state.
- Prep plan, prep history and rollover date are separated by business workspace.

## No SQL migration
This release uses the existing KitchenOps data model/browser workspace and needs no Supabase SQL migration.
